Waarom staat de tekst in de blokken horizontaal gecentreerd?

De tekst in de blokken staat horizontaal gecentreerd omdat deze van het element div zijn en in de css code kan je zien dat ze voor het div element de tekst centreren.

Ga na dat een property vertical-align:center geen effect heeft. Waarom eigenlijk niet?

Omdat het element geen inline of inline-block is.

Wat doet vertical-align dan wel?

Het zorgt voor de verticale uitlijning van de elementen. Vertical align middle zorgt ervoor dat de tekst in het midden geplaatst wordt. En het heeft geen invloed op de inhoud van het element.

Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren?

Doormiddel van line-heigt te gebruiken in de css code.

Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren?

Table cells wordt vaak gebruikt als truc. Je maakt hierbij gebruik van een tabel als soort van container en daarin centreer je de inhoud verticaal, dit doe je doormiddel van verticale uitlijning.

Waarom krijgt de regel voor class 'tall' voorrang?

Omdat de specificiteit van de klasse tall.
